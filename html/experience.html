User Experience

What have I learned about UX design?

The most important idea I have picked up through my classes on User Experience is that UX is an iterative process that shouldn’t be a one and done thing. Teams should always be actively looking on how to improve their product and UX is a continuous process that can be used in all stages of the development of their product.

How has my ideas about UX changed?

Entering into the UX program, I had a solid idea that User Experience and User Interface Design was the same thing, but realized very quickly that UI design was just a small part of UX and that their was so much more to learn about UX that I would have never thought related to UX.

How is UX useful to me?

As a Computer Science major, I typically work on projects that involve some sort of physical aspect, whether that be a UI or hardware of some sort. I have noticed that I spend more time coding than worrying about how the end user will use my product and how they will interact with the UI or hardware. After taking UX classes, I learned very quickly that I had to always should have the end user in mind and how they want to use the app along with how I want them to use the app. With this thought process, I provide better software and products that not only benefits society, but also improves my candidacy for future jobs and projects.
